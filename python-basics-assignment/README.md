# Basics of Python

##  Overview
This repository contains the solutions for the **Basics of Python** assignment completed using **Google Colab**. It covers fundamental Python programming concepts required for Data Science and Machine Learning.

##  Objectives
- Learn Python syntax
- Understand variables and data types
- Practice conditional statements
- Implement loops
- Work with functions
- Handle exceptions
- Perform file operations

##  Technologies Used
- Python 3
- Google Colab
