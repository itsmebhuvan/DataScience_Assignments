# Exploratory Data Analysis (EDA)

## Overview
This repository demonstrates data exploration and visualization techniques before building Machine Learning models.

- Data Cleaning
- Missing Values
- Outlier Detection
- Data Visualization
- Correlation Analysis

## Libraries

- NumPy
- Matplotlib
- Seaborn

## Tools
Google Colab

## Outcome
Understand dataset characteristics before model development.
